# PrestaShop Multi Blog Module

This is a basic PrestaShop module that adds a multi-language blog with SEO friendly URLs. It is aimed at PrestaShop 9 but works with recent PrestaShop versions.

## Features

* Manage posts from the administration panel
* Store posts in multiple languages
* SEO fields (slug, meta title and description)
* Friendly route `/blog/your-slug`
* Displays latest posts on the homepage

## Installation

1. Create a `psmultiblog` directory containing the files from this repository.
2. Compress the directory into `psmultiblog.zip`.
3. In the PrestaShop back office go to **Modules Manager > Upload a module** and upload the ZIP file.

After installation, use the **Configure** button to manage blog posts.
